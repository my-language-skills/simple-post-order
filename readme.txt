This plugin change the default post order in post-front.

== Description ==

This plugin change the default post order in post-front. From time order to alphabetical.

Based on: https://wpdevshed.com/guide-re-ordering-blog-posts/

https://github.com/my-language-skills/simple-post-order

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/simple-post-order` directory, or install the plugin
through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Access the plugins menu through the Clear Images menu page to use the plugin.
